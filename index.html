Goomovers

Acerca del Proyecto

Goomovers es una aplicación móvil diseñada para facilitar el proceso de mudanzas urbanas. Inspirada en plataformas como Didi o Uber, permite a los usuarios solicitar servicios de mudanza desde su celular, conectándolos con transportistas verificados en tiempo real.

Problemática

Las personas enfrentan dificultades para encontrar servicios de mudanza rápidos, confiables y asequibles. El proceso suele ser informal, sin garantías de calidad, seguridad o cumplimiento de horarios. Esto representa una oportunidad para digitalizar el sector.

Justificación del Problema

Goomovers busca ofrecer una solución tecnológica que profesionalice el servicio de mudanza, mejorando la experiencia tanto del cliente como del transportista. Su implementación permitirá mayor transparencia, eficiencia y acceso a una red segura de prestadores de servicio.

Objetivo Principal

Desarrollar una aplicación móvil que facilite el proceso de mudanzas urbanas, conectando a los usuarios con transportistas verificados de manera eficiente, segura y en tiempo real.

Objetivos Específicos

  • Proyectos: Planificar el desarrollo de una aplicación móvil de servicio de mudanzas, aplicando métodos de gestión de proyectos para garantizar el cumplimiento de plazos, requisitos y calidad.
  • Matemáticas: Aplicar los conocimientos adquiridos para poder realizar los cálculos matemáticos que requiera la aplicación.
  • Programación: Desarrollar las habilidades y conocimientos básicos necesario s para la generación del código de la aplicación y creación de las funciones básicas..
  • Bases Datos: Diseñar la estructura de una base de datos relacional con el propósito de gestionar de forma correcta la información recolectada de los usuarios.
  • P. Computacional: Aplicar los conceptos básicos del comportamiento de los sistemas computacionales con el fin de automatizar los procesos de la aplicación mediante algoritmos eficientes.
